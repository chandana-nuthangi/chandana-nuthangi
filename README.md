## 🧠 Featured project

### [Telugu NER — Transfer Learning for a Low-Resource Language](https://github.com/chandana-nuthangi/telugu-ner-thesis)
Fine-tuned DistilBERT-multilingual for Telugu Named Entity Recognition.
**F1 = 0.78**, within 2 points of published SOTA (IndicNER), using a **2.7× smaller
model** trained on **10% of the data**. Deployed as a Streamlit app with PDF entity
extraction. Includes an honest error analysis on out-of-domain text.
`PyTorch` `HuggingFace Transformers` `Streamlit` `NLP`
